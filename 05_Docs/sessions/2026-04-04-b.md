- Added `edgeBrightnessThreshold` to `RectangleDetector.Configuration`
- Replaced hardcoded `brightness > 100` checks with configurable threshold
- Added `edgeSensitivity: Double` to `SnapPointsManager` (0.0-1.0)

- Invalidates snap cache on sensitivity change, triggers re-detection
- Added slider in `SnapOptionsView` with Low/Med/High labels

### Cookbook
- Added two patterns: Debounced CIFilter Preview Cache (#22), Z-Order Overlay Dimming (#23)

## Decisions
- **Tab grouping:** Crop (spatial) / Effects (processing) / Export (output) — workflow-stage based
- **Quality & Resize:** Always visible in Export tab (no collapse) — used on every export
- **Global blur default:** Regions apply to all images. Per-image opt-out or custom override.
- **Sidebar width stays at 420px** — user preference, not reducing
- **Sensitivity mapping:** Linear 0→200, 1→30 brightness threshold — simple and effective
