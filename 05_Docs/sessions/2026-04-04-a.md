# Session: 2026-04-04a — v1.5 Implementation: Unified Crop/Blur Tool

## Goal
Implement the 4-wave plan from last session: unified crop/blur canvas with live preview, performance cache, and pixelate support.

## Progress

### Wave 1-3: Unified Canvas + Sidebar + Gestures
- Removed `EditorTool` enum and `currentTool` segmented picker
- `BlurEditorView` now always renders (under crop overlay for natural dimming)
- Added `isBlurDrawingEnabled` toggle in sidebar "Blur Regions" section header
- `B` key toggles blur drawing mode (preserves muscle memory)
- `Escape` deselects blur region or exits drawing mode
- Crop handles always on top (z-order: blur → crop overlay → crop handles)
- Blur regions outside crop area naturally dimmed by crop overlay
- Deleted `BlurRegionsCropPreview` (replaced by always-visible `BlurEditorView`)
- Wrapped unused `CropSectionView.swift` in `#if false`

### Wave 4: Pixelate Preview + Performance Cache
- Enabled all 4 blur styles in picker (pixelate was hidden)
- Pixelate live preview via `BlurPreviewCache` using `CIPixellate` (same filter as export)
- Cache-miss shows purple placeholder with spinner (~100ms until CIPixellate renders)
- `BlurPreviewCache`: pre-renders single composite `NSImage` at display resolution
  - Debounced 100ms, invalidates on region/intensity/image changes
  - Blur regions use cached composite on hit, SwiftUI `.blur()` fallback on miss
- Fixed pixelate intensity: removed 30% floor (`0.3 + intensity * 0.7` → `intensity`), 0% now means no effect
- Initial pixelate approach (double `scaleEffect`) didn't work — SwiftUI optimizes transforms away

### Bug Fix
- Pixelate intensity slider at 0% still showed pixelation — `effectivePixelateScale` had a 30% minimum floor from when pixelate was export-only. Removed floor so 0% = original image.

## Decisions
- **Blur drawing activation:** Toggle + B key (confirmed last session). Matches drawing app conventions.
- **Z-order:** Blur under crop overlay — regions outside crop are naturally dimmed (no badges needed)
- **Pixelate preview:** CIFilter via cache only (no SwiftUI-only fallback). Purple placeholder on cache miss is acceptable for ~100ms.
- **Pixelate intensity floor removed:** Was 30% min, now 0%. Users expect 0% = no effect with interactive slider.
- **Global blur regions deferred to v1.6** — blur should apply to all images by default with per-image opt-out
