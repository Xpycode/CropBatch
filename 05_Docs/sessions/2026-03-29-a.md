# Session: 2026-03-29

## Goal
Spec out Grid Split feature — user request to split images into NxN grid tiles and save as separate files.

## Progress

- Explored current sidebar UI structure and patterns (toggle-in-header, collapsible sections)
- Explored export/naming pipeline (ExportCoordinator, ExportSettings.outputURL, batchCrop flow)
- Designed feature spec with ASCII mockups, data model, pipeline placement, naming convention
- Saved full spec to `05_Docs/SPEC-grid-split.md`

## Decisions
- Grid split goes **after** Crop + Corner Mask, **before** Resize + Watermark in pipeline
- Uses "last tile absorbs remainder" for non-even divisions (industry standard)
- Naming: `{original}_{row}_{col}.ext` (1-indexed) — sorts correctly in Finder
- UI: own collapsible section with toggle, same pattern as Snap/Watermark
- Range: 1-10 for both rows and columns
- Model lives on `ExportSettings` (it's an export-time operation)
