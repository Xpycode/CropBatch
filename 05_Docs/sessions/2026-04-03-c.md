# Session: 2026-04-03c тАФ v1.5 Planning: Unified Crop/Blur Tool

## Goal
Research and plan the two highest-impact v1.5 features: live blur preview in crop mode and unified crop+blur tool.

## Progress

### Research Phase
- Deep-dived into full blur/crop architecture: data models, state managers, canvas views, gesture handling, export pipeline
- Researched 4 approaches for live blur preview:
  1. **Optimized SwiftUI `.blur()` with clipping** (current approach, improve in place)
  2. **Single CIFilter composite preview** (best overall тАФ reuses `ImageCropService.applyBlurRegions()`)
  3. **Metal `layerEffect` shader** (overkill for this use case)
  4. **NSVisualEffectView overlay** (wrong tool тАФ blurs behind window, not image content)
- Key finding: **live blur preview already exists** in `BlurRegionOverlay.previewContent` (line 296 of BlurEditorView.swift) тАФ it just only shows in blur mode

### Architecture Analysis
- Blur regions stored in normalized coords (0-1) relative to original image
- `BlurEditorView` has full interaction: draw, move, resize, live preview
- `BlurRegionsCropPreview` is a separate read-only view showing dashed outlines in crop mode
- Segmented picker in sidebar switches between two completely separate canvas states
- Export pipeline (Transform тЖТ Crop тЖТ Blur тЖТ ...) is untouched by this change

### Implementation Plan Created
Full plan at: `.claude/plans/fizzy-sparking-squid.md`

**4 waves:**
1. **Unified Canvas** тАФ Replace `BlurRegionsCropPreview` with real `BlurEditorView`, always render both crop + blur overlays
2. **Sidebar Unification** тАФ Remove segmented picker, show blur as collapsible section with drawing toggle
3. **Gesture Integration** тАФ Toggle + B key activation (confirmed by user), z-order for natural crop dimming of out-of-bounds regions
4. **Polish** тАФ Pixelate live preview, cached composite performance optimization

## Decisions
- **Blur drawing activation:** Toggle button in sidebar + B keyboard shortcut (over Option+drag or both). Preserves muscle memory, matches drawing app conventions.
- **Z-order strategy:** Blur renders under crop overlay тЖТ regions outside crop are naturally dimmed (replaces red X / orange triangle badges)
- **No pipeline changes needed** тАФ only view layer changes, data model and export untouched
