## Decisions
- Grid Split placed after Corner Mask, before Resize in pipeline
- GridSettings on ExportSettings (export-time operation, not crop setting)
- Tiles processed sequentially within each image task (text watermark thread safety)
- Last tile absorbs remainder pixels (industry standard)
- `processImageThroughPipeline` as shared pipeline — single source of truth for all export paths
- `NSBitmapImageRep` over `NSImage.lockFocus` for thread-safe offscreen rendering
- Dead sidebar structs moved to `_Unused/` rather than deleted
