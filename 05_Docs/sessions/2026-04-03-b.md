## Progress

### Production Hardening (from code review)

### Bug Fixes
- **Image flash on thumbnail switch** — `scaledImageView` downscale branch missing `.resizable()`, full-res image rendered at native pixel dimensions on cache miss. Added `.resizable().aspectRatio()` + guarded on `viewSize > 0`.
- **Blur export coordinates shifted** — Pipeline applied blur before crop, baking coordinates at original-image positions. Reordered to Transform → Crop → Blur, remapping regions via `applyingTransform` + `relativeToCrop`.
- **Corner radius didn't auto-set PNG** — Added `onChange` to save/restore format and lock picker when corner radius is on.
- **ColorPicker not opening** — Three watermark color pickers (text, shadow, outline) had empty labels with `.labelsHidden()` inside Form, collapsing the click target. Added proper labels and explicit frame size.
